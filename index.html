📦 Instalación en Termux

Guía paso a paso 
1. termux-setup-storage
2. pkg update -y && pkg upgrade -y
3. pkg install git nodejs ffmpeg imagemagick yarn -y
4. git clone https://github.com/Deylin-Eliac/Kirito-Bot-MD
5. cd Kirito-Bot-MD
6. yarn install
7. yarn start

🔁 Reiniciar:
cd Kirito-Bot-MD
yarn start

📷 Nuevo QR:
cd Kirito-Bot-MD
rm -rf kiritoSession
yarn start

📡 Segundo plano:
npm i -g pm2
pm2 start index.js
pm2 save
pm2 logs
